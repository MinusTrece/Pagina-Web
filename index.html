Inicio

El proyecto se enfoca en la problemática del uso excesivo de antibióticos químicos y su impacto ambiental en Ecuador. El objetivo general es desarrollar una página web para reducir el abuso de estos medicamentos y mejorar la calidad de vida de las personas mediante la información y capacitación sobre medicamentos alternativos. La metodología incluye la colaboración de médicos especialistas y se llevará a cabo en un plazo de 5 meses. La página web se mantendrá actualizada para mantener a la población informada sobre los nuevos medicamentos alternativos disponibles. El resultado esperado es una página útil y funcional para promover un estilo de vida saludable sin la necesidad de recurrir a tratamientos químicos.

Diagnostico

Objetivo general:

• El objetivo general de esta página web es proporcionar información detallada sobre enfermedades y tratamientos alternativos utilizando medicamentos naturales, a través de una interfaz fácil de usar y navegable desarrollada en lenguaje de programación HTML.

Desarrollar:

Desarrollar una página web informativa utilizando HTML como lenguaje de programación.

Informar

Proporcionar información detallada sobre enfermedades y tratamientos alternativos utilizando medicamentos naturales.

Crear:

Crear una interfaz fácil de usar y navegable para los visitantes de la página.

Justificacion

Este proyecto investiga la preferencia entre tratamientos convencionales y alternativos (naturales) para aumentar el conocimiento sobre los productos naturales y mejorar el aprovechamiento de estos recursos. Se busca abordar la interculturalidad y la medicina integrativa para mejorar el acceso a tratamientos para una gran porcentaje de la población. Se planea crear una página web informativa para educar a la población sobre el uso de medicamentos naturales en Ecuador.
